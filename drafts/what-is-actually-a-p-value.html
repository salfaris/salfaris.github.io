Significance testing: What is actually a p-value?

Posted on: Tue 18 May 2021

If you search "what is a p-value" on Google, you will be bombarded with various answers – some are simple, some are practical, and some have nice diagrams. Unfortunately, it is quite difficult to find an explanation that is mathematically rigorous and this will be the aim of my writing today. To talk about the p-value rigorously, we would have to first discuss the idea of significance testing.

So what is significance testing?

Significance testing is a formal procedure for comparing observed data with a claim, called the hypothesis, the truth of which is being assessed. The hypothesis is usually a statement about a parameter. We will see that the results of significance testing is expressed in terms of a probability which measures how well the data and the hypothesis agrees.

Consider the subset \(\Theta \subset \mathbb{R}^d\) which will be the space of parameters of interest – we call this the parameter space. Now let \(\mathbf{X} = (X_1, \ldots, X_n)\) be a vector of random variables such that \(X_i \sim f(x; \theta)\), for some density function \(f(x; \theta)\) whose parameter satisfies \(\theta \in \Theta\). The main point is that we assume that \(X_i\) is drawn from the same distribution, and for simplicity, we can further assume that they are independent (hence, i.i.d).

The idea now is that we have two classes of candidates for the parameter \(\theta\) above – \(\Theta_0\) and \(\Theta_1\) – which are disjoint subsets of \(\Theta\) and we want to decide which class should \(\theta\) belong to for our particular observed data. This leads us towards the idea of the null hypothesis \(H_0: \theta \in \Theta_0\) and alternative hypothesis \(H_1: \theta \in \Theta_1\). The next question is how to make this decision?

The test statistic

The magic tool we need is called the test statistic \(t(\mathbf{X}) = t(X_1, \ldots, X_n)\) which is simply a computable function of the sample data. It is itself a random variable whose distribution is known when \(H_0\) is true. If on the other hand \(t(\mathbf{X})\) takes extreme values, then we would doubt whether \(H_0\) is true.

Definition (p-value): Suppose now that we have the observations \(\mathbf{x} = (x_1, \ldots, x_n)\), so that the observed value of the test statistic \(t(\mathbf{X})\) is then \(t(\mathbf{x})\). Then the p-value of our test above is the probability obtaining a test statistic at least as extreme as the observed one, assuming \(H_0\) is true. That is, in a one-sided test,

$$ p = \mathbb{P}(t(\mathbb{X}) \geq t(\mathbb{x}) | H_0) $$

and in a two-sided test,

$$ p = \mathbb{P}(|t(\mathbb{X})| \geq t(\mathbb{x}) | H_0). $$
